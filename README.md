# 04-react-query

This is one of the projects I created during my studies. It's a web app that
helps you find a movie by keyword. It interacts with the TMDB service. It
features keyword search, page pagination, and a modal window displaying detailed
movie information.

## How to use

1. Copy this repository
   ```
   git clone (SSH or HTTPS key)
   ```
2. Install dependencies
   ```
   npm install
   ```
3. Start the server
   ```
   npm run dev
   ```
4. Or complete the project construction
   ```
   npm run build
   ```
